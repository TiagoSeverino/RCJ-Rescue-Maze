# RCJ-Rescue-Maze
Software used by my team (MazeRunners) in Portuguese Robotics Open 2017 Rescue Maze, where we got 6th Place.

# Hardware

Processing Units

* 1x Raspberry Pi 3

Sensors

* 5x SRF04
* 1x CMPS10
* 1x Line Sensor
* 1x Dip Switch

Actuators

* 4x Motors
* 1x Servo Motor
